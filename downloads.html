German Noun Gender Trainer (GNGT)

Windows

GNGT v1.0 - 3.1MB

Supported versions of the Windows operating system are Windows XP, Windows Vista and Windows 7.

Ubuntu Linux

GNGT v1.0 - 0.1MB (For i386-compatible PCs)

To install:

Download the package.

Extract package contents.

Right click the "gngt_1.0_i386.deb" file then select "Open With Ubuntu Software Center".

Wait for Ubuntu Software Center to start then select install.

Some packages might need to be downloaded and installed. This will be done automatically by Ubuntu Software Center. Download size varies between different systems.

This was tested with Ubuntu 10.10

Maemo Linux

GNGT v1.0 - 0.1MB

To install:

Download the package.

Extract package contents.

Send "gngt_1.0.0_armel.deb" to your device.

Open it and follow on-screen instructions.

This was tested with Maemo5 PR1.3.

Other Linux Distributions

To use GNGT on other distributions, you can download the source code and compile it (after installing Qt4).

GNGT can be used on any distribution and architecture supported by Qt4.

Symbian

GNGT v1.0 - 0.1MB

Qt v4.7.1 - 12.2MB

To install GNGT on your Symbian powered smart phone, you need to install Qt. The tested version is v4.7.1.

Devices compatible with GNGT are those compatible with Qt. In other words, most Nokia smart phones released after 2006 and some smart phones of other brands. More specifically, Symbian powered smart phones that run S60 versions 3.1, 3.2 or 5.0. Plus models that run Symbian^3 and (probably) later versions of Symbian.

For more info about Nokia smart phones, visit Forum Nokia.

If you don't know if your device fits the previous description, simply download and install GNGT and Qt, and if you got no error messages during installation then your device is probably compatible with GNGT.

Note for Nokia E-Series users: you need to change certain settings to be able to install the program, in case an error message appeared during installation.
This setting can be changed in the device as follows: Open the application manager, then select: Settings > Software installation > All.
